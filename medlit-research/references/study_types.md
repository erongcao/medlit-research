# 医学研究类型与评价要点

## 证据等级金字塔

```
        系统综述/Meta分析
              ↑
         RCT (随机对照试验)
              ↑
          队列研究
              ↑
         病例对照研究
              ↑
          横断面研究
              ↑
           病例报告
              ↑
           专家意见
```

## 各研究类型的适用场景

### 1. 随机对照试验 (RCT)
- **目的**: 评价干预措施疗效
- **金标准**: 因果关系推断
- **关键特征**: 随机化、对照、盲法
- **适用**: 新药、新疗法评价
- **局限**: 成本高、外推性有限

### 2. 队列研究
- **目的**: 评价暴露与结局关联
- **类型**: 前瞻性、回顾性
- **适用**: 危险因素研究、预后研究
- **优点**: 可计算发病率、RR
- **局限**: 时间长、失访偏倚

### 3. 病例对照研究
- **目的**: 罕见疾病危险因素
- **适用**: 罕见病、长潜伏期疾病
- **优点**: 快速、低成本
- **局限**: 回忆偏倚、无法计算发病率

### 4. 横断面研究
- **目的**: 患病率调查
- **适用**: 疾病负担、现状描述
- **优点**: 快速、经济
- **局限**: 无法确定时间顺序

### 5. 诊断试验研究
- **目的**: 评价诊断准确性
- **指标**: 敏感性、特异性、PPV、NPV
- **适用**: 新诊断方法评价

### 6. 系统综述/Meta分析
- **目的**: 综合现有证据
- **适用**: 证据综合、临床实践指南
- **质量**: 取决于纳入研究质量

## 偏倚风险评估

### 选择偏倚
- 随机序列生成
- 分配隐藏
- 组间基线可比性

### 实施偏倚
- 盲法（受试者、干预者）
- 方案偏离

### 测量偏倚
- 盲法（结局评估者）
- 结局测量标准化

### 随访偏倚
- 失访率
- 失访原因
- 意向性治疗分析

### 报告偏倚
- 选择性报告结局
- 发表偏倚

## 关键评价问题

### 所有研究类型通用
1. 研究问题是否明确？
2. 研究设计是否适合研究问题？
3. 方法学质量如何？
4. 结果是否可靠？
5. 结论是否得到数据支持？
6. 是否适用于我的患者/情境？

### 证据适用性 (External Validity)
- 研究人群是否与我的患者相似？
- 干预措施是否可行？
- 获益-风险比是否可接受？
- 价值观和偏好是否匹配？
