# 医学研究报告规范

## CONSORT - 随机对照试验报告规范

### 核心清单 (25项)

#### 标题和摘要
- [ ] 标题能识别为随机对照试验
- [ ] 结构式摘要包含：背景、方法、结果、结论

#### 引言
- [ ] 科学背景和原理解释
- [ ] 具体目的或假设

#### 方法
- [ ] 试验设计（如平行、析因）
- [ ] 试验开始后对方法的改变及原因
- [ ] 纳入和排除标准
- [ ] 数据收集的场所和地点
- [ ] 干预措施的详细描述（足够重复）
- [ ] 主要和次要结局指标的定义
- [ ] 样本量计算及主要指标
- [ ] 随机化：序列生成、分配隐藏机制、实施
- [ ] 盲法：对象、干预者、评估者
- [ ] 统计方法：主要/次要指标分析方法、亚组分析、缺失数据处理

#### 结果
- [ ] 参与者流程图（推荐CONSORT流程图）
- [ ] 各组基线人口学和临床特征
- [ ] 各组纳入分析的受试者数量及排除原因
- [ ] 主要和次要结局结果（效应估计及精确度）
- [ ] 二元结果：同时报告相对和绝对效应
- [ ] 辅助分析：其他分析和亚组分析
- [ ] 伤害：各组不良事件

#### 讨论
- [ ] 主要结果解释（考虑研究假设、潜在偏倚、多重比较）
- [ ] 可推广性（外部有效性）
- [ ] 总体证据解释（与其他研究的关系）

#### 其他
- [ ] 试验注册号和注册机构
- [ ] 资助来源和其他支持
- [ ] 可获取的完整试验方案

## PRISMA - 系统综述和Meta分析报告规范

### 核心清单 (27项)

#### 标题
- [ ] 标题明确标识为系统综述或Meta分析

#### 摘要
- [ ] 结构式摘要：背景、目的、资料来源、纳入标准、参与者、干预、评价方法、结果、局限性、结论、系统综述注册号

#### 引言
- [ ] 理论基础
- [ ] 目的：使用PICO描述

#### 方法
- [ ] 方案和注册
- [ ] 纳入标准：PICO、研究设计、时间、报告特征、语言
- [ ] 信息来源：数据库、注册库、其他来源、最后检索日期
- [ ] 检索：至少一个数据库的完整检索策略
- [ ] 研究选择：筛选过程、纳入排除标准
- [ ] 数据提取过程
- [ ] 数据项目：提取的数据列表
- [ ] 单个研究的偏倚风险：评估工具、风险评价方法
- [ ] 效应指标：合成的主要结局指标
- [ ] 合成方法：统计方法、异质性处理
- [ ] 报告偏倚风险评估
- [ ] 补充分析：敏感性分析、亚组分析方法

#### 结果
- [ ] 研究选择：PRISMA流程图
- [ ] 研究特征：每项研究的特征引用表
- [ ] 单个研究的偏倚风险：评价结果展示
- [ ] 单个研究结果：每项研究的干预效果估计
- [ ] 合成结果：Meta分析森林图
- [ ] 报告偏倚：评估结果（漏斗图）
- [ ] 补充分析：敏感性分析、亚组分析结果

#### 讨论
- [ ] 证据总结
- [ ] 局限性：综述水平和证据水平
- [ ] 结论：证据优势、对实践和研究的启示

#### 资金
- [ ] 资金来源及作用

## STROBE - 观察性研究报告规范

### 核心清单 (22项)

#### 标题和摘要
- [ ] 摘要中使用结构化格式
- [ ] 摘要中提供背景和目的、方法、结果、结论

#### 引言
- [ ] 背景/原理
- [ ] 目的：具体目标，包括任何预设假设

#### 方法
- [ ] 研究设计：呈现研究设计的关键要素
- [ ] 研究背景：地点、时间、数据收集日期
- [ ] 参与者：纳入排除标准、来源、选择方法
- [ ] 变量：明确所有结局、预测、潜在混杂、效应修饰因子
- [ ] 数据来源/测量：每个变量的数据来源和详细测量方法
- [ ] 偏倚：描述潜在偏倚来源
- [ ] 研究规模：样本量确定方法
- [ ] 定量变量：分组方法、原因
- [ ] 统计方法：描述所有统计方法（充分详细）

#### 结果
- [ ] 参与者：参与者流程图（推荐）
- [ ] 描述性数据：参与者特征、缺失数据
- [ ] 结局数据：报告主要结局的受试者数量
- [ ] 主要结果：非调整和调整关联、精确度（CI）
- [ ] 其他分析：亚组分析、敏感性分析

#### 讨论
- [ ] 关键结果：总结关键结果，参考研究目标
- [ ] 局限性：讨论研究局限性、潜在偏倚、不精确性
- [ ] 解释：结合研究目标、局限性、多重比较、类似研究
- [ ] 可推广性：讨论结果的外部有效性

#### 其他
- [ ] 资金：资金来源和作用

## CASP - 批判性评价技能计划

### CASP RCT清单 (11个问题)

1. 研究问题是否集中？
2. 分配是否随机？
3. 组间是否可比？
4. 是否对受试者、干预者、评估者设盲？
5. 除干预外各组是否接受相同治疗？
6. 效应大小？
7. 精确度如何（95%CI）？
8. 益处是否大于危害和成本？
9. 结果对我的患者是否有帮助？

### CASP 系统综述清单 (10个问题)

1. 综述是否提出明确的问题？
2. 是否检索了适当的数据库？
3. 是否进行了充分的文献检索？
4. 纳入标准是否明确？
5. 是否对纳入研究进行了严格评价？
6. 提取数据的方法是否可靠？
7. 各研究的结果是否相似？
8. 总体结果是什么？
9. 结果的精确度如何？
10. 结果是否可应用于我的患者？
